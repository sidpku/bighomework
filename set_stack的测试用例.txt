默认d=2

[0,0,0,0,0,0]

['t', 0, 0, 0, 0, 't']
['t', 0, 0, 0, 0, 't']
['t', 't', 't', 't', 't', 't']

fields2：（凸字形，凹角处形状不变）
[0,0,0,0,0,0]
